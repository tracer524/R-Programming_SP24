基于POI数据的北京市喜茶连锁店选址情况分析¶

案例背景： 近年来，国内现制茶饮行业迅猛发展，产生了如喜茶、乐乐茶、奈雪的茶等诸多知名品牌。而如何选址，对茶饮门店可谓是重中之重，它与门店全生命周期的经营发展密切相关。合适的选址可以让门店充分利用周边资源，带来高人流与成高交量；而选址有误却有可能让原本备受欢迎的茶饮品牌遇冷，甚至迎来关门闭店的结局。本案例以国内最知名的茶饮品牌之一喜茶为切入点，收集喜茶在北京市的门店以及周边POI数据，对喜茶连锁店在北京的选址展开研究。

这个案例的学习目标包括：

1、学习数据读入、数据清洗等数据预处理方法

2、学习描述分析、聚类分析等数据分析方法

3、学习数据分析与业务实际相结合的商业分析模式

任务五 竞对协同分析¶

将餐饮POI中提供与喜茶同质但具有一定差异性的产品的POI，定义为喜茶的【竞对】，例如，咖啡厅、酒吧、茶座等POI。将餐饮POI中提供的产品类似喜茶产品的辅助者的POI，定义为喜茶的【协同】，例如，中餐厅、小吃快餐店、外国餐厅、蛋糕甜品店等POI。此处以【竞对】为例进行分析，从门店竞对数、竞对评分、价格、评论几个维度，对喜茶门店的竞对情况进行描述分析。

任务六 经营优势分析¶

本案例定义一个新指标，用于衡量喜茶在线下的经营优势，具体指标定义如下：

喜茶线下经营优势 = log(喜茶门店人流 / 门店周边POI平均人流)

绘制这一指标的分布直方图与箱线图，对喜茶线下经营优势情况进行详细分析。

##poi变量：poi数目，poi最多的行业，poi平均评分，poi平均评论数，poi平均价格，poi平均距离
def get_avg(all_list, is_price=False): 
    price_list = []
    for price in all_list:
        if (is_price and pd.isna(price)==False and price.isdigit()) or (is_price==False and pd.isna(price)==False):
            if float(price) > 0:
                price_list.append(float(price))
    return np.mean(price_list)

store_name_list = xicha['name'].tolist()
poi_store_info = pd.DataFrame(columns=["name","poi_cnt","poi_max_tag","poi_rate_avg", "poi_comment_avg", "poi_price_avg", "poi_dist_avg"])
for name in store_name_list:
    temp_df = poi[poi['store_name']==name]
    poi_cnt = temp_df.shape[0]
    poi_max_tag = list(temp_df['tag_detail_1'].value_counts().sort_values(ascending=False).index)[0]
    poi_rate_avg = get_avg(temp_df['overall_rating'])
    poi_comment_avg = get_avg(temp_df['comment_num'])
    poi_price_avg = get_avg(temp_df['price'], True)
    poi_dist_avg = get_avg(temp_df['distance'])
    temp_dict = {"name":name,"poi_cnt":poi_cnt,"poi_max_tag":poi_max_tag,"poi_rate_avg":poi_rate_avg, "poi_comment_avg":poi_comment_avg, "poi_price_avg":poi_price_avg, "poi_dist_avg":poi_dist_avg}

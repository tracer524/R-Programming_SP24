本案例共包含4个数据集。各数据集具体情况如下所示：

数据集1：全文.txt。包含了《三体》三部的文本数据集。

数据集2：全文人数.txt。人工清洗的较为全面的《三体》人物名称数据集。一行一个人物及其别名。

数据集3：情感互动矩阵.csv。人工清洗的包含了《三体》人物情感互动数据集，共包含4列，分别为角色对所处段落、段落情感得分、角色1名称、角色2名称。
数据清洗方式为：通过情感词典判断段落的情感得分情况，情感得分为正数则记为1，负数记为-1。获得每一段的情感得分后，提取段落出场任务对，并清洗成上述数据集。

数据集4：社区发现结果.csv。数据集包含笔者某次社区划分的结果，共包含2列，分别是角色名、角色归属的社区ID。
